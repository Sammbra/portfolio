The Last of Us

The Last of Us website was created as part of a college assignment focused on designing an engaging single page website for a favorite TV series. The goal was to build a dynamic layout where content updates within the same page, using dropdown menus in specific sections to keep the experience interactive and easy to navigate. The project was designed with both desktop and mobile versions in mind, ensuring that the same interactions and structure work smoothly across devices. The website includes the most important sections, starting with a strong hero section that sets the tone for the rest of the experience and features a dynamic fireflies animation created in After Effects.

Project overview

Date: May, 2023
Scope: UI, Motion Effects
Programs: Adobe Photoshop, Adobe Illustrator, Figma, Adobe After Effects

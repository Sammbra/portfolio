Symphony

Symphony is a website project created as part of my Bachelor’s degree. The project involved designing and developing a website for an imaginary music streaming service. The website was designed in Figma and includes custom illustrations made specifically for it and music themed icons. A key feature is the implementation of both light and dark themes, which adapt to the user’s operating system settings and can also be manually switched via an icon button. The selected theme is saved and consistently applied on future visits. The website was built using HTML, CSS, and JavaScript, following Google’s Material Design guidelines. This project allowed me to combine UI design and frontend development while focusing on usability, theming, and responsive interface behavior.

Project overview

Date: April, 2022
Scope: Illustration, Logo, UI, Web development
Programs: Adobe Photoshop, Adobe Illustrator, Figma, Visual Studio Code
